Splash Screen

Technical Description

The Splash Screen features a background video overlayed with text, a button, and a loading animation. Upon loading, the page automatically redirects after a few seconds. The layout is responsive, and Flexbox is used to ensure that all elements adjust correctly across various screen sizes. The animation for the text is achieved using CSS keyframes, combined with @keyframes, transform, and animation properties to create a smooth fade-in effect. The three-dot loading animation is implemented using span elements and the nth-child pseudo-class, ensuring a clean and smooth visual appearance.


Accessibility

To ensure the page is accessible, units like px, vh, %, and rem are used for the layout. The lang attribute is set to "en", and ARIA roles are applied to elements. The page is compatible with all modern web browsers, offering a consistent experience for users with visual impairments.

Volunteer Page

Technical Description

Technical Description The Volunteer page is designed to encourage potential volunteers to join and contribute to various causes. The layout is fully responsive, utilizing Flexbox for efficient content arrangement across different screen sizes. Key sections include a volunteer sign-up form with interactive buttons and a dynamic statistics section where the counters animate as the user scrolls, providing an engaging visual effect. JavaScript powers the animated counters, and elements like buttons and dropdowns feature smooth hover effects for better interactivity. A fade-in animation enhances the user experience, drawing attention to important sections. The page also includes expandable review cards that users can interact with, revealing more details about past volunteer experiences. Form validation is built into the registration process to ensure accurate data entry. Additionally, the page is designed to be visually appealing while maintaining simplicity and usability.


Accessibility

The Volunteer page is built with accessibility in mind, ensuring a seamless experience for all users, including those with disabilities. Browser-supported units like px, vh, %, and rem are used to achieve a responsive design that adapts to different devices and screen sizes. The lang attribute is set to "en", and alt text is provided for images to assist visually impaired users relying on screen readers. The page leverages ARIA labels and semantic HTML elements like div, section, and form to create a structured and accessible layout. The inclusion of interactive elements such as the review cards and statistics counters follows accessibility best practices, ensuring that they are easily navigable by keyboard and compatible with assistive technologies.

Content Page

Technical Description

The Content page consists of icons placed between horizontal lines, which serve a stylistic purpose but are not interactive. The layout is responsive, leveraging Flexbox to ensure the page adapts to different screen sizes. The page is simple and uses no complex properties, making it lightweight and efficient. The content is divided into sections, and each section contains relevant information with appropriate images. The icons are visually appealing and provide a clean look without adding interactive elements.


Accessibility

The page is designed to be fully accessible with the use of responsive units like px, % and rem. The lang attribute is set to English, ensuring that screen readers can interpret the content correctly. The alt text is added for images to provide context for visually impaired users, and ARIA labels are applied to ensure smooth navigation.

Challenges and Lessons Learned


One of the main challenges I faced during the development of this project was integrating all the elements seamlessly into a responsive layout. Although Flexbox proved to be a valuable tool for managing the layout, I encountered difficulties while fine-tuning the animations and form validations. Additionally, collaborating with my team on the Volunteer page helped me learn how to effectively manage JavaScript to create interactive statistics.

Compliance


Our team has ensured that the web pages adhere to web publishing regulations, including ensuring that no offensive or illegal content is displayed. We also made sure that the site complies with accessibility standards and that all external content, such as images, is properly credited. The website respects intellectual property rights and does not infringe upon any copyrights.
