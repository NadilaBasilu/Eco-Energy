Home Page

Webpage starts with a video with overlaying text, using top and left property. The 5 images under the our sponsor section have scale up hover effect.The next 5 images also have this effect, but it also have a section of text that slides in from left. Here I used transform property and :hover pseudo-class. under the above mentioned text there is also a see more button linking to relevent sections of the articles, which has a color change hover effect. the next 4 sections doesn't have any note worthy properties used. the links of the featured articles section have a color changing hover effect. using @keyframes rule to add a scroll effect to the our sponsors, statistics, voices of our members and featured articles sections as well as about us img and mission image. the scale and opacity increase as the user scrolls up and vise versa. As our tutor adviced 2 arrows are added to scroll to the top and to scroll to the bottom. This web page is responsive, becuase of the use of flexbox method.


Footer

The logo used has a transparent back-ground. There are 5 sections Articles, arranged using css grid. The links under them have a text color change hover effect.


Accessibility

used px and % as a unit of measurement. this page can be viewed in any web browser. set the lang property to english. and added clsses for div, section and alt for img properties.
